- Price of each Pineapple = 12.5 Bdt

| Quantity     | Discount | 
|--------------|----------|
| 1 to 9       | 5%       |
| 10 to 19     | 10%      |
| 20+          | 15%      |

- Add tips as it is
- If Home Delivery = Yes, add 40 BDT, else nothing
- Shop Location

| Location     | Tax      | 
|--------------|----------|
| Dhaka        | 20%      |
| Chittagang   | 15%      |

- Calculation Hint
 - 12.5 x 5 = 62.5
 - 62.5 x 0.95 = 59.375
 - 59.375 + 20 = 79.375
 - 79.375 + 40 = 119.375
 - 119.375 * 1.15 = 137.28125
